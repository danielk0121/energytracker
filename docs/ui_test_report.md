## 3. 발견된 잠재적 UI/UX 개선 사항

### [3-1] 사진 분석 피드백 부족
- **현상**: 사진을 선택하지 않은 상태에서 `사진 분석 시작` 버튼을 눌러도 사용자에게 아무런 경고나 알림(예: "먼저 사진을 선택하세요")이 표시되지 않음.
- **제안**: 파일 선택 전 버튼을 비활성화(`disabled`)하거나, 클릭 시 알림 팝업 추가 권장.

### [3-2] 테이블 전환 시 레이아웃 튀는 현상
- **현상**: `월별 목록`과 `일자별 목록` 간 전환 시 행 개수 차이로 인해 페이지 높이가 급격히 변함.
- **제안**: 최소 높이(`min-height`)를 설정하여 화면 전환 시의 시각적 흔들림 최소화.
