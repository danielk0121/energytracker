## 1. 헤더 레이아웃 구조

헤더 영역은 브라우저 상단 우측에 고정된 버튼 그룹과 중앙의 메인 제목으로 구성되어 있습니다.

### [1-1] 버튼 그룹 영역 (`.theme-toggle-container`)
- **역할**: 다크모드 전환 및 메뉴(설정) 접근 버튼을 감싸는 컨테이너
- **스타일 특징**: `position: absolute`, `top: 10px`, `right: 10px`를 사용하여 상단 우측에 배치됨

### [1-2] 아이콘 버튼 (`.icon-btn`)
- **역할**: 모든 버튼에 공통으로 적용되는 시각적 스타일
- **스타일 특징**: `border-radius: 50%`로 원형 디자인이며, 호버 시 배경색 변화(`transition`)가 적용됨

### [1-3] 제목 영역 (`h1`)
- **역할**: "에너지트래커" 서비스 명칭 표시
- **스타일 특징**: `color: var(--primary-blue)`, `text-align: center`로 중앙 정렬됨

| 클래스 이름 | 설명 |
